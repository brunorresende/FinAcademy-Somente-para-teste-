Sobre a FinAcademy

A FinAcademy é a materialização prática de um projeto acadêmico de Front-End, desenvolvido ao longo de um semestre na disciplina de Trabalho Interdiciplinar I (TI I) da PUC Minas.

Nossa missão fundamental é viabilizar e democratizar o acesso à educação financeira, um tema de importância vital e tão recorrente na sociedade atual. Em um cenário econômico complexo, ferramentas digitais e intuitivas são essenciais para capacitar o usuário.

O projeto foi concebido para ir além do básico, utilizando HTML, CSS e JavaScript para criar uma plataforma robusta e interativa. Demonstramos a aplicação de técnicas de Desenvolvimento Web Moderno e a implementação de um sistema de gerenciamento de dados (CRUD) integrado a quizzes e módulos de aprendizado.
