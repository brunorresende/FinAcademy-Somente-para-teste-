# FinAcademy

Este aplicativo tem como objetivo promover a educação financeira de forma acessível e prática. Ele oferece dicas rápidas para ajudar o usuário a economizar, investir e organizar seu dinheiro no dia a dia, além de uma calculadora financeira completa que realiza simulações de juros simples e compostos, financiamentos, empréstimos e parcelamentos. 
Complementa-se com um mini-curso teórico dividido em módulos, abordando temas como orçamento, tipos de investimento, inflação e risco versus retorno, com linguagem clara, exemplos cotidianos e quizzes interativos.

## Instruções de utilização 

O site é direcionado para aprendizado e prática do letramento financeiro para brasileiros. Nele você pode cadastrar dados sobre seu perfil econômico para servir de base para outras funções. Nele, existe uma aba para estudar conteúdos e realizar exercícios, destinado a aprendizagem do zero e também há uma calculadora financeira, para realização de atividades práticas e servir de ajuda em situações reais. Além disso, dentro do site, existe um algoritmo para analisar o risco de determinados investimentos, de forma a proporcionar uma experiência completa para desenvolvimento financeiro, mesmo para quem não detêm nenhum conhecimento prévio.
